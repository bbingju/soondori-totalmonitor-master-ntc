1.2 Frame Format for the Internal

  • Octet 2개(STR: 0xFE, END: 0xFD)가 예약되어 있음
  • Frame의 시작은 STR, 끝은 END

  • 각 field는 다음과 같음:
    Slot ID
    1 octet. 0 - 3까지 값을 가짐. 0xFF 이면 모든 슬롯을 의미함.
    CMD
    1 octet. 아래에서 구체적으로 설명함.
    Payload length
    1 octet. payload의 최대 크기는 0xFF (256) bytes.
    Payload
    N1
    CRC
    2 octets. Slot ID부터 데이터 끝까지의 구간을 계산함.

1.2.1 Internal CMDs

  1. INTERNAL_CMD_BOARD_TYPE_REQ, (0x01)
  2. INTERNAL_CMD_RESET, (0x02)
  3. INTERNAL_CMD_SLOT_ID_REQ, (0x03)
  4. INTERNAL_CMD_TEMPERATURE_REQ, (0x04)
  5. INTERNAL_CMD_TEMPERATURE_STATE_REQ, (0x05)
  6. INTERNAL_CMD_ADC_REQ, (0x06)
  7. INTERNAL_CMD_THRESHOLD_REQ, (0x07)
  8. INTERNAL_CMD_THRESHOLD_SET, (0x08)
  9. INTERNAL_CMD_RELAY_REQ, (0x09)
  10. INTERNAL_CMD_RELAY_SET, (0x0A)
  11. INTERNAL_CMD_COMPENSATE_CONSTANT_REQ, (0x0B)
  12. INTERNAL_CMD_COMPENSATE_CONSTANT_SET, (0x0C)
  13. INTERNAL_CMD_COMPENSATED_REQ, (0x0D)
  14. INTERNAL_CMD_COMPENSATED_SET, (0x0E)
  15. INTERNAL_CMD_CORRECTION_NTC_CON_TABLE_CAL, (0x0F)
  16. INTERNAL_CMD_CORRECTION_NTC_CON_TABLE_REQ, (0x10)
  17. INTERNAL_CMD_CORRECTION_NTC_CONSTANT_REQ, (0x11)
  18. INTERNAL_CMD_CORRECTION_NTC_CONSTANT_SET, (0x12)

uint16_t crc16_ccitt(const uint8_t *buf, int len)
{
        register int counter;
        register uint16_t crc = 0;
        for( counter = 0; counter < len; counter++)
                crc = (crc<<8) ^ crc16tab[((crc>>8) ^ *(uint8_t *)buf++)&0x00FF];
        return crc;
}
